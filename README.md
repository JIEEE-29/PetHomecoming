# 宠物归家系统

一个基于 `Python 标准库 + SQLite + 原生 HTML/CSS/JS` 的轻量 Web 系统，覆盖：

- 人员注册、登录、资料审核
- 摄像头拍照、本地图片处理与上传
- 宠物分类、状态识别
- 宠物评论、救治联系、领养/认领申请

## 运行

```bash
cd E:\Codex\pet-homecoming
python server.py
```

访问地址：

- [http://127.0.0.1:8000](http://127.0.0.1:8000)

## 默认管理员

- 用户名：`admin`
- 密码：`admin123`

## 数据位置

- 数据库：`pet_homecoming.db`
- 上传目录：`uploads/`
